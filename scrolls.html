he content that the user will navigate through using scrollspy needs to be inside of a container decorated with data-spy='scroll', and its data-target attribute set to the ID set on the nav element (again using CSS selector syntax). If you want to use scrollspy for the navigation of the entire page, say for a frequently asked questions (FAQ) page, you can use those attributes on the body element. Finally, the style of the containing element needs to have position:relative in order to ensure the tabs highlight correctly as the user scrolls.
Inside of the nav element, you'll add an unordered list. The ul element needs to have both nav and navbar-nav classes added to it. Inside of there you will add the links that will act as the navigation links to the scrollspy content. Each link will point to the ID of the content's container, again making sure to use CSS selector syntax. The link for the first data to be displayed should have the active class added.
